為 Ngnix 建立自行簽章的憑證

2013年10月17日

Linux SSL Nginx

必要條件

產生憑證需要 OpenSSL 類別庫，請先確認系統是否已安裝 OpenSSL。

若尚未安裝，可透過以下方式安裝：

  • Mac OS X 使用 Homebrew 安裝

    $ brew install openssl

  • Windows 可到此網站下載安裝檔：OpenSSL for Windows

  • Ubuntu/Debian Linux

    $ sudo apt-get install openssl

  • RedHat/CentOS Linux

    $ sudo yum install openssl
